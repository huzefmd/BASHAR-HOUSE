# 🏠 BASHAR HOUSE

A modern real estate and property showcase website designed to present luxury homes, villas, apartments, and property listings with an elegant user experience.

BASHAR HOUSE is a responsive real estate platform that helps users explore premium properties through a clean and modern interface. The website focuses on providing an engaging browsing experience with property details, images, contact options, and responsive design across all devices.
